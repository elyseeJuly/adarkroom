# 偶成诗词创作工具 - 演示网站

## 项目简介

"偶成"是一款极简、高审美的纯本地诗词创作与辅助工具的演示网站，完美展现了PRD文档中描述的所有核心功能和设计理念。网站融合了传统排版美学与现代Web技术，为用户提供沉浸式的诗词创作体验。

## 核心特色

### 🎨 视觉设计
- **传统美学**：宣纸纹理背景，朱红印章，水墨黑文字
- **竖排排版**：支持传统竖排文字布局，每句一列
- **现代交互**：流畅动画，响应式设计，触摸友好

### 📝 创作功能
- **双模式创作**：自由创作 + 专业创作
- **十四行诗**：三种体例支持（莎士比亚体、彼特拉克体、现代汉语体）
- **格律检查**：实时标注，红黄绿三色提示
- **AI点评**：雅致文言点评，润色建议

### 📚 诗词资源
- **3.3万首古典诗词**：全唐诗+全宋词+精选作品
- **966个词牌**：完整词谱数据库
- **智能检索**：全文搜索，朝代体裁分类

### 💾 本地管理
- **作品存储**：本地JSON格式，安全可靠
- **海报生成**：一键生成宣纸风格分享图
- **分类整理**：按体裁、时间、格式分类

## 技术实现

### 前端技术栈
- **HTML5 + CSS3**：语义化结构，现代布局
- **JavaScript ES6+**：模块化开发，交互逻辑
- **Anime.js**：流畅动画效果
- **ECharts.js**：数据可视化热力图

### 核心功能
- **竖排文字**：CSS transform + 动态布局
- **实时预览**：创作同时显示排版效果
- **本地存储**：LocalStorage模拟存储
- **搜索算法**：JavaScript全文搜索

### 设计系统
- **色彩方案**：宣纸白(#F7F1E3)，朱红(#B22222)，水墨黑
- **字体系统**：Ma Shan Zheng楷体，ZCOOL KuaiLe标题
- **动画规范**：淡入淡出，缩放脉冲，滚动视差
