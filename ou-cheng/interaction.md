# 偶成诗词创作工具 - 交互设计

## 核心交互流程

### 1. 创作模式切换
- 顶部模式切换器：自由创作 ↔ 专业创作
- 平滑过渡动画，保持用户上下文

### 2. 自由创作模式交互
- **多行输入框**：
  - 自动去除首尾空格
  - 实时字数统计
  - 支持换行和段落
- **十四行诗专区**：
  - 三种体例选择器（莎士比亚体、彼特拉克体、现代汉语十四行）
  - 实时格律检查：红色标注违规，黄色标注轻微问题
  - 当前体例名称显示
- **创作心声**：
  - 可折叠的灵感记录区域
  - 支持多行文本输入
- **AI点评**：
  - 自动生成60字内文言点评
  - 雅致的语言风格
- **实时预览**：
  - 竖排模式：每句一列，右到左排列
  - 横排模式：传统书信式布局
  - 宣纸纹理背景

### 3. 专业创作模式交互
- **体裁选择**：
  - 近体诗按钮 → 显示格律模板
  - 词牌按钮 → 显示分类选择（小令/中调/长调）
- **词牌选择器**：
  - 966个词牌数据库
  - 搜索和筛选功能
  - 模板预览（仅显示平仄和韵位）
- **韵书切换**：
  - 词林正韵（默认）
  - 平水韵
  - 中华新韵
- **格律检查**：
  - 绿色：严格合律
  - 黄色：可通押
  - 红色：不合律
- **笔名管理**：
  - 下拉选择已有笔名
  - 新建笔名功能

### 4. 作品管理交互
- **作品列表**：
  - 首句摘要 + 日期显示
  - 横/竖排版图标
  - 心声预览（前20字）
- **操作手势**：
  - 左滑删除
  - 长按生成宣纸海报
- **海报生成**：
  - 自动生成高雅分享图片
  - 宣纸纹理背景
  - 随机淡印章效果

### 5. 寻章摘句交互
- **搜索功能**：
  - 标题、作者、正文全文搜索
  - 模糊匹配支持
- **结果展示**：
  - 诗题 + 作者 + 朝代
  - 完整正文显示
  - 关键词高亮
- **灵感热力图**：
  - 朝代/体裁分布可视化
  - 交互式图表

### 6. 经典赏读交互
- **分类浏览**：
  - 按体裁分类（先秦、乐府、近体诗、词、曲）
  - 按朝代分类
- **阅读体验**：
  - 线装古籍模拟
  - 上下翻页手势
  - 极美留白设计
- **内容展示**：
  - 诗经、楚辞、乐府诗集
  - 唐宋金元明清代表作
