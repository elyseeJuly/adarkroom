# 偶成诗词创作工具 - 项目概述

## 项目简介
"偶成"是一款极简、高审美的纯本地诗词创作与辅助工具，融合传统排版美学与现代AI技术，完全离线可用。产品slogan"佳句本天成，妙手偶得之"体现了对自然天成诗词创作的追求。

## 网站结构

### 1. 主页 (index.html)
**目标**：展示产品核心理念和美学价值
- **Hero区域**：
  - 传统书法背景图像
  - 产品名称"偶成"和品牌印章
  - 核心标语"佳句本天成，妙手偶得之"
  - 简洁的功能介绍
- **功能预览区**：
  - 四大核心功能卡片展示
  - 创作工具、作品管理、诗词检索、经典赏读
- **美学展示**：
  - 竖排文字示例
  - 宣纸纹理效果演示
  - 传统色彩搭配

### 2. 创作页面 (create.html)
**目标**：展示核心创作功能
- **模式切换器**：自由创作 ↔ 专业创作
- **自由创作区**：
  - 多行输入框
  - 十四行诗专区（三种体例）
  - 创作心声记录
  - AI点评展示
- **专业创作区**：
  - 体裁选择（近体诗/词牌）
  - 词牌选择器（966个选项）
  - 韵书切换
  - 格律检查标注
- **实时预览**：
  - 竖排/横排切换
  - 宣纸背景
  - 传统排版效果

### 3. 作品管理页面 (works.html)
**目标**：展示本地作品管理功能
- **作品列表**：
  - 卡片式布局
  - 首句摘要 + 元信息
  - 排版方向图标
- **操作功能**：
  - 搜索和筛选
  - 作品编辑
  - 删除功能
  - 海报生成演示
- **统计信息**：
  - 作品数量
  - 创作时长
  - 体裁分布

### 4. 经典赏读页面 (classics.html)
**目标**：展示古典诗词阅读功能
- **分类导航**：
  - 按体裁分类（先秦、乐府、近体诗、词、曲）
  - 按朝代分类
- **诗词展示**：
  - 线装古籍风格
  - 竖排文字布局
  - 优雅分页
- **搜索功能**：
  - 全文搜索
  - 作者筛选
  - 朝代筛选
- **热力图**：
  - 朝代分布可视化
  - 体裁统计图表

## 技术架构

### 前端技术栈
- **HTML5**：语义化结构，无障碍访问
- **CSS3**：现代布局技术（Flexbox/Grid）
- **JavaScript ES6+**：模块化开发，交互逻辑
- **动画库**：Anime.js实现流畅动画效果
- **着色器**：Shader-park创建宣纸纹理效果

### 核心功能实现
- **竖排文字**：CSS transform + JavaScript动态布局
- **格律检查**：本地算法实现，颜色标注系统
- **数据存储**：LocalStorage模拟本地存储
- **搜索功能**：JavaScript全文搜索算法
- **海报生成**：Canvas API绘制分享图片

### 响应式设计
- **桌面优先**：针对iPad优化
- **平板适配**：768px - 1023px断点
- **手机兼容**：≤767px简化布局
